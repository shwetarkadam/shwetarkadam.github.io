#+title: Vim
Vim is a text editor. Vim stands for Vi iMproved.
Vim consists of different modes.

- Normal mode :: the default mode, used for editor commands
- Visual mode :: can highlight areas of text and editor commands will run on that highlighted text
- Insert mode :: similar to editing in most editors
